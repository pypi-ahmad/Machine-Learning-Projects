# Dataset Sources

## Key

| Symbol | Meaning |
|--------|---------|
| **K** | Kaggle (requires `kaggle.json` API key) |
| **TV** | torchvision built-in (auto-download, no credentials) |
| **GYM** | Gymnasium environment (no download needed) |

## Manual Steps

| Dataset | Action Required |
|---------|----------------|
| All **K** datasets | Place `kaggle.json` in `~/.kaggle/` before first run |
| CelebA, COCO, Places2 | Multi-GB downloads — ensure sufficient disk space |
| E-commerce Behavior | ~6 GB — may take significant time to download |
