## Quick Start

```bash
git clone https://github.com/pypi-ahmad/Machine-Learning-Projects.git
cd Machine-Learning-Projects

python -m venv .venv
.venv\Scripts\Activate.ps1          # Windows PowerShell
# source .venv/bin/activate         # Linux / macOS

pip install -r requirements.txt
```

Run any project:

```bash
python "Classification/Adult Salary Prediction/pipeline.py"
python "Time Series Analysis/Stock Price Forecasting/pipeline.py"
python "Reinforcement Learning/Lunar Landing/pipeline.py"
```

## Generator

All 315 pipelines are generated by `_overhaul_v2.py` which maps each project to the correct model family. Each pipeline auto-downloads its data at runtime (no manual dataset management).

```bash
python _overhaul_v2.py           # regenerate all 315 pipelines
```

See [MIGRATION_PLAN.md](MIGRATION_PLAN.md) for the complete family-by-family model audit.
